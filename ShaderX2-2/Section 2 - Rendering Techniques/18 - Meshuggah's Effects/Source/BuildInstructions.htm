In order to successfully build the executable run the following scripts...

1. makeDll.bat under .\3rdPartyDlls to create an encrypted dll archive called dlls.cry

2. makeData.bat under .\Data to create an encrypted data archive called data.cry

These two files will be part of the executable's resource. If the content of any of these archives changes run the script(s) again and delete the .res file under both .\Debug and .\Release of the main app. This enforces recompilation of the resource the next time you do a build.
